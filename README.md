# mvc_demo
一个练手做的图书增删查改功能[基础]

### 第一周
使用servlet+mysql+jsp实现
### 第二周
修改为Ajax+JS实现,使用js对表单进行序列化、ajax将前端数据发送给servlet

### 第三周
修改为使用Jquery+json实现

### 第四周
实现多页列表的显示

## 使用：
在自己Tomcat的server.xml文件中添加MySQL连接池配置
以及在context.xml添加：
<ResourceLink global="sysdemo" name="sysdemo" type="javax.sql.DataSource" />
